# Toddler quickstart (2 minutes)

## First time only

1. Install the app you were given (`Toddler-Claude-Setup.msi`). Click through.
2. Open **Toddler Claude**.
3. You will see four boxes to fill in. The app will ask for:
   - Your Claude password (via the browser — no typing passwords into the app).
   - Your GitHub account (via the browser too).
   - A key from Fly.io that the person who set this up gave you.
4. Pick the project from the list.
5. Click **Finish setup**.

## Every day

1. Open **Toddler Claude**.
2. Click the big green **Start working** button.
3. Wait ~20 seconds. A black terminal appears.
4. Type what you want in English. Example:
   > Add a search box to the home screen and make pressing Enter do the search.
5. Claude thinks for a bit, then makes the changes. When it says it's done, type:
   > Commit and open a pull request.
6. Claude gives you a link to the pull request. Close the app. You're done.

## If something goes wrong

- **Red error in the app** → click "Stop and discard", then click "Start working" again. 99% of problems are fixed by restarting.
- **Claude won't stop talking / is stuck** → click "Stop and discard". Start again.
- **You typed something sensitive by mistake** (like a password) → click "Stop and discard" immediately. The machine will be destroyed. Ask the owner to help you rotate.
- **You can't start** → the machine might be out of budget. Message the owner.

## What NOT to do

- Do **not** paste passwords, API keys, or credit card numbers into Claude. Ever.
- Do **not** install random tools. Claude already has what it needs.
- Do **not** try to click on files starting with `.env`.

That's it. Have fun.
